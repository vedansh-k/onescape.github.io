# OneScape Website Template

## Folder Structure
- index.html - Homepage
- about.html - About Us
- services.html - Services
- portfolio.html - Projects
- contact.html - Contact Form
- css/style.css - Custom styling
- js/main.js - JavaScript
- images/ - Add your images here

## Hosting Instructions
1. Create a GitHub account
2. Create a new repo: onescape.github.io
3. Upload all files
4. Go to Settings > Pages > Select branch 'main' and root '/'
5. Site will be live at https://onescape.github.io 

## GoDaddy Domain Setup
1. Go to GoDaddy DNS
2. Add CNAME record: www → onescape.github.io
3. Add A records:
